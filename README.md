# YAP (Yet Another Pipeline)
Pipeline(s) for mapping and cluster-level aggregation of single nucleus methylome and multi-omic datasets.
Technologies supported:
- snmC-seq(1/2/3)
- snmCT-seq (mC + RNA)
- any NOMe treated version of the above
